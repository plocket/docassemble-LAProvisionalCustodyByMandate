# Louisiana Provisional Custody By Mandate

Allow a parent in Lousiana to give a mandatary the ability to make decisions for the parent's children about such things as medical care and education.

Part of the [Document Assembly Line US Delegation of Parental Rights project](https://github.com/SuffolkLITLab/50StateDelegationOfParentalRights/issues/1).
